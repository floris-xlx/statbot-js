# RateLimitHeaders

Rate limit headers for the API response.

**x-ratelimit-limit**integerrequired

Maximum number of requests that can be made in the current time window.

**Example:** `10`

**x-ratelimit-remaining**integerrequired

Number of requests remaining in the current time window.

**Example:** `5`

**x-ratelimit-reset**integerrequired

How many seconds until the rate limit resets.

**Example:** `60`

**retry-after**integer

If max requests are reached, the time in seconds to wait before making a new request.

**Example:** `30`

RateLimitHeaders

```
{  "x-ratelimit-limit": 10,  "x-ratelimit-remaining": 5,  "x-ratelimit-reset": 60,  "retry-after": 30}
```
