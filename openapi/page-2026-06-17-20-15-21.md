# Invite

An invite.

**id**integerrequired

Statbot ID of the invite.

**Example:** `456`

**memberId**string<snowflake>required

Discord ID of the inviter.

**channelId**string<snowflake>required

Discord ID of the channel the invite is for.

**code**stringrequired

Discord invite code.

**Example:** `abc123`

**uses**integerrequired

Number of times the invite has been used.

**maxUses**integerrequired

Maximum number of times the invite can be used.

**maxAge**integerrequired

Maximum age of the invite in seconds.

**isTemporary**integer<boolean-integer>required

Whether the invite grants temporary membership (0 = false, 1 = true).

**isVanity**integer<boolean-integer>required

Whether the invite is a vanity URL (0 = false, 1 = true).

**isActive**integer<boolean-integer>required

Whether the invite is currently active (0 = false, 1 = true).

**label**stringrequired

Custom label for the invite.

**Example:** `My Invite`
